# TechyProfessor — Portfolio for Sarju S

This repository hosts a lightweight, responsive portfolio website for **Sarju S**, also known as **TechyProfessor** — an educator, startup mentor, and innovation enabler based in Kerala, India. The content is curated from publicly visible details on the provided LinkedIn profile.

## Stack

- Static HTML5
- Modern CSS (custom properties, responsive grid)
- Vanilla JavaScript for data-driven sections and theme management

No build tooling is required; open `index.html` in a browser.

## Structure

```
techyprofessor/
├── index.html      # Markup and section anchors
├── styles.css      # Visual system, layout, and responsive behavior
├── script.js       # Content data, DOM rendering, theme + nav toggles
└── README.md
```

## Content Highlights

- Hero story highlighting Sarju’s community impact
- Impact, journey, volunteer, and recognition sections populated with LinkedIn data
- Projects grid referencing live platforms (WordPress site, GitHub org, dashboards, admissions tooling)
- Certifications, publications, and recommendation quotes
- Accessible contact section with Netlify-ready form and social links

## Usage

1. Clone or download this directory.
2. Open `index.html` locally or host the folder on any static site service (GitHub Pages, Netlify, Vercel, etc.).

## Customization Tips

- Update the color palette in `styles.css` to match brand preferences.
- Extend the `data` object in `script.js` with additional experiences or assets.
- Replace the avatar placeholder in the hero with an actual portrait when available.
